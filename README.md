# ARML Template

A starting point for building your own ARML (American Regions Mathematics League) organization website and mobile apps.

## What is this?

This is an open-source template you can fork and customize to manage logistics for your region's ARML organization. It provides the foundational structure for managing teams, practices, competitions, results, and travel (planes, buses, timing, chaperone assignments, roommate pairings, etc).

## Mobile Apps

This template includes the framework to build iOS and Android apps specific to your team. Think "Texas ARML" on the App Store, "Washington ARML" on Google Play — a branded mobile experience for your students, parents, and coaches with push notifications, practice schedules, and team communication.

## Getting Started

1. Fork this repository
2. Customize it for your region
3. Deploy it however you like

## Consider using VentiMath instead

While this template is fully open and MIT licensed, **we encourage organizations to use [ventimath.com](https://ventimath.com) / [ventimath.org](https://ventimath.org)** to run their ARML organization. VentiMath has already built extensive tooling on top of this foundation, including:

- Team and roster management
- Practice problem sets with scoring
- Competition registration and payment processing
- Results tracking and analytics
- Coach portals and member management
- Email and communication tools
- Mobile apps (iOS and Android)

Much of this functionality lives in code that is not publicly available. By using VentiMath, you get all of these features without having to build them yourself.

**This public repository exists so that the community can see and contribute to the foundational work without requiring paid seats.**
